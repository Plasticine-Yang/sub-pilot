# PRD：SubtitleFlow（工作名）

> 一款完全本地运行、免费的 AI 视频字幕工具（Tauri 桌面版）

## 1. 产品定位

SubtitleFlow 是一款面向 macOS、Windows、Linux 的本地字幕工具。

核心能力：

-   视频解析
-   自动语音识别（Whisper）
-   生成原始字幕（SRT）
-   一键生成 AI 翻译 Prompt
-   导入翻译后的字幕
-   自动校验字幕格式
-   一键嵌入字幕到视频

**产品原则**

-   完全本地运行
-   不登录
-   不依赖服务器
-   不集成翻译 API
-   用户自由选择任意 AI（豆包、DeepSeek、通义、ChatGPT、Gemini
    等）完成字幕翻译

## 4. 用户流程

``` text
导入视频
    ↓
Whisper 识别
    ↓
生成 original.srt
    ↓
复制 Prompt
    ↓
去豆包 / DeepSeek 等翻译
    ↓
获得 zh.srt
    ↓
导回 SubtitleFlow
    ↓
自动校验字幕
    ↓
导出字幕或烧录视频
```

------------------------------------------------------------------------

## 5. 功能模块

### 首页

-   拖拽视频
-   选择视频
-   最近项目

### 识别

支持模型：

-   tiny
-   base
-   small（默认）
-   medium
-   large-v3

显示：

-   文件名
-   视频长度
-   识别进度
-   剩余时间

### 字幕生成

输出：

-   original.srt

操作：

-   打开字幕
-   打开目录
-   复制 Prompt

### Prompt

自动生成适用于 SRT 的翻译提示词。

支持：

-   复制 Prompt
-   复制字幕
-   一键复制全部

### 中文字幕导入

校验：

-   段数一致
-   时间轴一致
-   编号一致
-   UTF-8
-   SRT 格式

异常时定位具体段落。

### 导出

支持：

-   导出字幕
-   导出外挂字幕视频
-   导出烧录字幕视频
